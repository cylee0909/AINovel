# 第81章 大梁初到

## 【登场角色】
- **李衍**：抵达魏国新都大梁
- **墨鸢**：观察魏国技术和建筑
- **魏国官员**：接待外来士子
- **大梁市民**：迁都后的新都居民
- **魏国士子**：对李衍既好奇又排斥

## 【承】
到达大梁，李衍被这座新兴都城震撼。大梁位于中原腹地，交通便利，魏惠王迁都于此，显露出争霸中原的野心。城市正在大规模建设：城墙高厚，宫殿宏伟，街道规划整齐。但李衍也看到问题：迁都耗费巨万，国库空虚；大兴土木征发民力，百姓怨声；旧都安邑的贵族不愿迁来，新旧势力矛盾。

## 【展】
李衍以“游学士子”身份入住驿馆。魏国官员例行接待，态度冷淡——魏国自视中原正统，对外来士子不如齐国热情。李衍不急于求见魏王，而是先了解魏国情势：拜访当地学者，观察市井民情，收集军政情报。他得知：魏国正与齐国紧张对峙，庞涓率军在前线；魏王刚愎自用，不听谏言；朝中分为主战派和主和派，斗争激烈。

## 【转】
李衍重点打听“九鼎”线索。通过接触魏国老学者，得知：“传说魏室藏有‘武鼎’，乃魏文侯时所得。但迁都后，不知是否运来大梁。”武鼎——听名字可能与军事相关。同时，墨鸢探查大梁建筑，发现异常：新王宫地基有特殊结构，似为隐藏某物而建。她设计探测装置，发现王宫地下有大型金属物体反应——很可能是鼎！

## 【落】
就在李衍探查时，魏国突发大事：庞涓从前线发回急报，称“齐军异动，恐攻魏”。魏王紧急召集群臣商议。主战派主张“先发制人，攻齐救赵”（此时赵国正被齐国威胁）；主和派认为“魏国新都未稳，不宜大战”。李衍知道，这是马陵之战的序幕。他面临抉择：是否预警魏国此战必败？根据阴符规则，重大战役结果难改，但预警可能减少伤亡。他决定尝试，但需巧妙方式。眉间疤痕微微发热，预警着即将到来的历史风暴。（**钩子**：李衍会如何预警？魏王会听吗？**伏笔埋下**：发现魏国可能有武鼎；马陵之战序幕拉开；李衍面临干预历史的选择；魏国内部矛盾显现。）
