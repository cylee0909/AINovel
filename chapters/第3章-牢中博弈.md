# 第3章：牢中博弈

暴雨过后的邯郸大牢，空气中弥漫着泥土与霉变的混合气味。李衍被两名狱卒架着，从水牢转移到了一间相对干燥的单人牢房。

“算你走运。”年长的狱卒解开他手腕上的麻绳，麻绳已经勒进皮肉，留下一圈紫黑色的淤痕，“赵将军吩咐了，给你换个地方。”

李衍活动着僵硬的手腕，目光扫过这间新牢房。三面土墙，一面木栅，地上铺着薄薄一层干草。比起水牢，这里至少能让人躺下。墙角放着一个陶碗，里面是半碗掺杂着麦粒的饼——虽然粗糙，但至少是能入口的食物。

他靠着墙壁坐下，背脊传来土墙的凉意。左眉上方的疤痕隐隐作痛，那是穿越时留下的印记。三天了，距离他预言匈奴入侵已经过去三天，赵奢再也没有出现。

“心理战术。”李衍低声自语。

他太了解这种手段了。现代审讯学中，冷处理是最基础的心理施压方式。赵奢在观察他，观察一个自称能预知未来的人在等待中的反应——是会焦躁不安，还是崩溃求饶，抑或是保持沉默。

李衍闭上眼睛，开始在脑海中构建“记忆宫殿”。

这是他在大学时期训练出的记忆技巧，如今成了他最大的依仗。智能手机早已没电，但里面的历史资料库以另一种形式存在于他的脑海深处。他“走”进宫殿的第一层，那里存放着《史记·匈奴列传》的全文。

“匈奴，其先祖夏后氏之苗裔也，曰淳维……”

文字在脑海中浮现，但他需要更具体的信息。公元前350年，赵成侯在位。他“走”向宫殿的第二层，那里存放着《战国策》和《资治通鉴》的相关记载。

“赵成侯二十一年，匈奴左贤王寇边，掠代郡、雁门……”

时间、地点、兵力。李衍在脑海中反复核对。历史记载往往简略，但他需要细节——三万骑兵如何分兵，左贤王的作战风格，匈奴各部落的集结规律。这些信息散落在各种史料中，需要他像拼图一样重新组合。

第四天清晨，牢房外传来沉重的脚步声。

李衍睁开眼睛，看到赵奢站在木栅外。这位赵国将领今天没有穿甲胄，只着一身深褐色麻布常服，但腰间依然挂着那柄缺口的青铜剑。他右脸上的刀疤在晨光中显得格外狰狞，从眼角一直延伸到下颌，像一条蜈蚣趴在脸上。

赵奢身后跟着一个满脸横肉的中年汉子，穿着皮甲，腰间挎刀，眼神凶戾如鹰隼。

“开门。”赵奢的声音低沉。

狱卒连忙打开牢门。赵奢走进牢房，居高临下地看着坐在地上的李衍。他的身材魁梧如熊，站在狭小的牢房里几乎挡住了所有光线。

“将军，此子来历不明，妖言惑众，当严刑拷问！”中年汉子抢先开口，声音粗粝如砂纸摩擦。

赵奢抬手制止，目光始终锁定在李衍脸上：“汝说三月后匈奴犯边，有何依据？”

李衍缓缓站起身。他的身体依然虚弱，但背脊挺得笔直。他知道，这是第一次真正的交锋。

“七月末，”他的声音平静而清晰，“匈奴左贤王将率三万骑南下，分三路进攻：主力攻代郡，偏师袭雁门，游骑扰云中。”

军司马——李衍判断那中年汉子的身份——冷笑一声：“胡言乱语！左贤王上月刚与我国签订盟约，献马百匹、牛羊千头，岂会背信弃义？”

“盟约是假，缓兵之计是真。”李衍迎上军司马的目光，“匈奴今春遭遇白灾，漠北草原大雪月余，牛羊冻死过半。若不南下劫掠，这个冬天至少有三成部落要饿死人。”

赵奢的眼神微动。

李衍继续道：“左贤王的长子呼衍术，正在阴山以北秘密集结兵力。此事可从匈奴俘虏口中验证——就说秋季狩猎，但秋季狩猎从不需要集结万人。”

牢房里陷入短暂的沉默。军司马还想反驳，赵奢却抬手制止了他。

“带匈奴俘虏来。”赵奢命令道。

半个时辰后，两个匈奴俘虏被押到牢房外的走廊。他们穿着破烂的羊皮袄，头发编成辫子，脸上涂着赭红色的颜料。看到赵奢时，两人的眼中都闪过恐惧——赵奢脸上的刀疤，正是与匈奴作战留下的。

审讯用匈奴语进行。赵奢亲自审问，他的匈奴语带着浓重的赵国口音，但足够交流。李衍听不懂具体内容，但从俘虏的表情和肢体语言中，他能判断出赵奢在问什么。

第一个俘虏跪在地上，浑身发抖，语无伦次。

第二个俘虏稍微镇定些，但眼神闪烁。

赵奢突然暴喝一声，右手按在剑柄上。那俘虏吓得瘫倒在地，嘴里吐出一连串急促的话语。李衍捕捉到几个关键词：“呼衍术”“集结”“狩猎”。

审讯持续了一刻钟。结束时，赵奢的脸色变得凝重。

他挥手让狱卒带走俘虏，转身看向李衍：“呼衍术确实在集结部落勇士，理由是秋季狩猎。”

军司马立刻道：“将军，秋季狩猎是匈奴常事，未必就是……”

“但他说集结了万人。”赵奢打断他，“往年秋季狩猎，最多不过三五千人。”

牢房里的气氛陡然紧张起来。军司马瞪大眼睛，似乎还想争辩，但看到赵奢的表情，又把话咽了回去。

赵奢走近一步，距离李衍只有三尺：“就算匈奴有异动，也可能是小规模骚扰，岂会如你所说三万大军？”

李衍知道，最关键的时刻到了。

他深吸一口气，缓缓道：“因为匈奴得到了燕国的暗中支持。”

“什么？”军司马失声叫道。

赵奢的瞳孔骤然收缩。

“燕国欲借匈奴之力牵制赵国，”李衍的声音压得很低，但在寂静的牢房里清晰可闻，“以便夺取中山国故地。燕王已经派密使与左贤王接触，承诺提供粮草和情报，换取匈奴在七月末发动进攻。”

“荒谬！”军司马怒道，“燕赵虽有龃龉，但同属华夏，岂会勾结胡虏？”

李衍没有理会军司马，目光直视赵奢：“将军应该知道，去年燕国在中山边境增兵三千。名义上是防备齐国，但实际上呢？中山国灭亡已有十年，其故地三分，赵国占其二，燕国占其一。燕王一直对赵国占据的中山城池耿耿于怀。”

赵奢的右手握紧了剑柄，指节发白。

李衍继续加码：“燕国密使名叫公孙操，是燕国大夫公孙贾的庶子。他三日前从蓟城出发，走北线经涿郡入匈奴地界。如果将军现在派人拦截，或许还能抓到证据。”

牢房里死一般的寂静。

军司马张了张嘴，却发不出声音。他看向赵奢，发现将军的脸色已经变得铁青。

赵奢突然转身，对军司马道：“你带一队骑兵，立刻往北线追查。若有燕国使者的踪迹，不惜一切代价拦截。”

“将军，这……”军司马还想说什么。

“执行命令！”赵奢的声音如寒冰。

军司马咬了咬牙，抱拳行礼，快步离开牢房。脚步声在走廊里渐行渐远，最终消失。

现在，牢房里只剩下赵奢和李衍两人。

赵奢缓缓转过身，目光如刀，一寸寸刮过李衍的脸。他的眼神复杂——有怀疑，有震惊，有警惕，还有一丝难以掩饰的……兴奋。

“汝究竟是何人？”赵奢的声音压得很低，几乎是从牙缝里挤出来的。

李衍知道，机会来了。

他没有直接回答，而是反问道：“将军可曾想过，为何赵国屡次北伐匈奴，却始终无法根除边患？”

赵奢眯起眼睛。

“因为赵国用错了方法。”李衍继续道，“匈奴是游牧民族，逐水草而居。赵国大军北上，他们便远遁漠北；赵国撤军，他们便卷土重来。如此反复，徒耗国力。”

“那依你之见，该如何？”

“以骑制骑。”李衍吐出四个字，“匈奴的优势在于机动，赵国的优势在于组织和装备。将军的胡服骑射是正确方向，但还不够。需要改良马具，训练专门的轻骑兵部队，建立边境预警体系，还要……”

他停顿了一下，观察赵奢的反应。

赵奢的呼吸变得急促。他脸上的刀疤在昏暗的光线下微微抽动，那是肌肉紧绷的表现。

“还要什么？”赵奢追问。

“还要改变战略思维。”李衍一字一句道，“不是被动防御，而是主动出击。不是等到匈奴南下才反击，而是在他们集结时就打散他们。不是只靠军事，还要用经济手段——控制盐铁贸易，让匈奴依赖赵国的物资，从而掌握主动权。”

赵奢沉默了。

他盯着李衍，眼神中的怀疑逐渐被另一种情绪取代——那是猎手发现珍贵猎物的眼神。

良久，赵奢缓缓开口：“这些……你是如何知道的？”

李衍没有回答这个问题。他知道，有些秘密现在还不能说。

“我是一个能助将军立不世之功的人。”他平静地说，“将军想要重振赵氏武勋，想要在对抗匈奴的战争中建立功勋，想要洗刷家族的耻辱。而我，能帮你做到这些。”

赵奢的右手松开了剑柄。他后退一步，重新打量李衍——这个穿着破烂麻衣、浑身伤痕、却眼神深邃如古井的年轻人。

“你想要什么？”赵奢问。

“自由。”李衍毫不犹豫，“还有证明自己的机会。”

“如果刚才说的都是真的，”赵奢缓缓道，“如果燕国真的勾结匈奴，如果七月末匈奴真的会大举入侵……那你就不只是证明自己那么简单了。”

李衍听出了言外之意。

他微微躬身：“愿为将军效劳。”

赵奢没有立刻回应。他在牢房里踱步，皮靴踩在干草上发出沙沙的声响。一步，两步，三步……他在思考，在权衡。

终于，他停下脚步。

“我会验证你所说的每一句话。”赵奢的声音恢复了军人的冷硬，“如果燕国密使的事情属实，我会向君上举荐你。但如果有一句虚言……”

他没有说完，但眼神已经说明了一切。

李衍点头：“明白。”

赵奢转身离开牢房。走到门口时，他停顿了一下，回头道：“从今天起，你的饮食会改善。好好养伤，我需要你活着。”

牢门重新关上，锁链发出哗啦的声响。

李衍缓缓坐回干草堆上，长长地吐出一口气。后背已经被冷汗浸湿，心脏在胸腔里剧烈跳动。

他成功了。

至少，成功了一半。

赵奢已经开始相信他，或者说，至少开始重视他提供的情报。接下来的关键，是军司马能否截获燕国密使。如果成功，他的地位将彻底改变；如果失败……

李衍摇摇头，甩开这个念头。

他重新闭上眼睛，进入记忆宫殿。这次，他“走”向存放军事技术的区域。改良马鞍、马蹄铁、马镫……这些在战国时代尚未普及的技术，将成为他献给赵奢的第一份礼物。

但在此之前，他需要先活下去。

牢房外传来脚步声，不是赵奢，也不是军司马。是狱卒，端着新的食物——一碗热腾腾的粟米粥，还有一块真正的麦饼。

李衍接过食物，感受着陶碗传来的温度。

他知道，这场博弈才刚刚开始。赵奢的信任还很脆弱，军司马的敌意依然存在，而更大的危机——匈奴入侵——正在三个月后等着他。

但至少，他有了立足之地。

有了在这个残酷时代，第一次真正意义上的……机会。

窗外，晨光渐亮。邯郸城在黎明中苏醒，市井的喧嚣隐约传来。而在遥远的北方，阴山脚下，匈奴的骑兵正在集结。

历史的车轮，已经开始转动。

而李衍，这个来自未来的穿越者，正试图用自己的双手，去改变它的轨迹。

哪怕只是一点点。
